Create `packages/modules/auth/src/__tests__/create-tenant.test.ts` (per D-06, D-03):

This file validates that `mock.module("../auth", ...)` correctly intercepts the `import { auth } from "../auth"` in `commands/create-tenant.ts`. This is the most critical pattern for all auth handler tests.

Structure:
1. Import `describe, test, expect, mock, beforeEach` from `"bun:test"`
2. Import `createMockContext` from `"../../../__test-utils__/mock-context"` (relative path from auth/__tests__ to modules/__test-utils__)
3. Import `assertResultOk, assertResultErr` from `"../../../__test-utils__/assert-result"`
4. Create `const mockCreateOrganization = mock(() => Promise.resolve({ id: "org-1", name: "Test Org", slug: "test-org" }))`
5. Call `mock.module("../auth", () => ({ auth: { api: { createOrganization: mockCreateOrganization } } }))`
6. Dynamically import: `const { createTenant } = await import("../commands/create-tenant")`
7. Write 3 tests in a `describe("createTenant", ...)` block:

**Test 1: "creates tenant and emits tenant.created event"**
- `const ctx = createMockContext({ userId: "user-1" })`
- `const result = await createTenant({ name: "My Company" }, ctx)`
- `assertResultOk(result)`
- `expect(result.data).toEqual(expect.objectContaining({ id: "org-1", name: "Test Org" }))`
- `expect(ctx.emit).toHaveBeenCalledWith("tenant.created", { tenantId: "org-1", createdBy: "user-1" })`
- `expect(mockCreateOrganization).toHaveBeenCalledTimes(1)`

**Test 2: "returns error when auth.api throws"**
- `mockCreateOrganization.mockRejectedValueOnce(new Error("Slug already taken"))`
- `const ctx = createMockContext()`
- `const result = await createTenant({ name: "Duplicate" }, ctx)`
- `assertResultErr(result, "Slug already taken")`

**Test 3: "auto-generates slug from name when slug not provided"**
- `const ctx = createMockContext()`
- `await createTenant({ name: "My Company!" }, ctx)`
- `expect(mockCreateOrganization).toHaveBeenCalledWith(expect.objectContaining({ body: expect.objectContaining({ slug: "my-company" }) }))`

Add `beforeEach(() => { mockCreateOrganization.mockClear(); })` to reset mock state between tests.

**CRITICAL validation:** If `mock.module("../auth")` does NOT intercept correctly (tests fail with DB connection errors), try alternative paths:
