**Step 2: Create `packages/ui/src/components/data-table-cards.tsx`**

Create the DataTableCards component following the UI-SPEC card layout specification. The component accepts a TanStack React Table instance and renders rows as cards.

```typescript
"use client"

import * as React from "react"
import type { Table as ReactTable, Column, Row, flexRender } from "@tanstack/react-table"
```

Import `flexRender` from `@tanstack/react-table`. Import `Card, CardContent` from `./card`. Import `Select, SelectContent, SelectItem, SelectTrigger, SelectValue` from `./select`. Import `cn` from `../lib/utils`. Import `ChevronDown` from `lucide-react`.

**Interface:**
```typescript
interface DataTableCardsProps<TData> {
  table: ReactTable<TData>;
  priorityCount?: number;        // how many priority columns to show (default: 3)
  onRowClick?: (row: TData) => void;
  renderActions?: (row: TData) => React.ReactNode;
  emptyMessage?: string;
}
```

**Column priority logic:**
- Read `column.columnDef.meta` as `{ priority?: number; cardHidden?: boolean }`
- Priority columns: `meta.priority >= 1 && meta.priority <= priorityCount`, sorted by priority ascending
- Detail columns: all columns where `meta.cardHidden !== true` AND (`meta.priority` is undefined OR `meta.priority > priorityCount`)
- Hidden columns: `meta.cardHidden === true` (e.g., actions column)

**Card layout per UI-SPEC:**
```
+--------------------------------------+
| [Priority 1 value]     [Status badge]|
| Priority 2 label: Priority 2 value   |
|                              [chevron]|
+--------------------------------------+
```

Implementation details:
- Each card is a `<Card>` with `<CardContent className="p-4">` (16px padding per UI-SPEC)
- Cards container: `<div className="space-y-4">` (16px gap per UI-SPEC)
- Priority 1 field: render with `text-sm font-semibold` (card title role per UI-SPEC)
- Priority 2+ fields: render as `<div>` with `<span className="text-xs text-muted-foreground">` for label and `<span className="text-sm">` for value
- Chevron icon (ChevronDown, 16px) in bottom-right, `text-muted-foreground`, rotates 180deg when expanded via `transition-transform duration-200` and `rotate-180` class
- Chevron has `aria-label="Show details"` when collapsed, `aria-label="Hide details"` when expanded
- Tap anywhere on the card toggles expand (via onClick on Card)
- Expanded state: separator line (`<div className="my-3 border-t" />`) between summary and detail
- Detail fields use same label/value pattern as priority 2+ fields
- Card cursor: `cursor-pointer`
- Card border: default shadcn Card border (1px solid border color)

**Sort dropdown (per D-09):**
- Render above the cards list if any column has `enableSorting: true`
- Use `Select` component with label "Sort by"
- Options: all columns where `getCanSort()` is true, display the header string
- On select: call `column.toggleSorting()` on the selected column
- Show current sort state in the trigger text

**Empty state:**
- When `table.getRowModel().rows.length === 0`, show `emptyMessage` or default "No results."

**Expand state management:**
- Use `React.useState<string | null>(null)` to track which row ID is expanded
- Toggle on card click: if already expanded, collapse; otherwise expand this row
