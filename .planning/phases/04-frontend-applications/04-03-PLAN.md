```typescript
      { path: "system", lazy: () => import("../routes/system/health") },
    ],
  },
]);
```

React Router 7 `lazy` routes expect named exports: `Component` (the page component), optionally `loader`, `ErrorBoundary`. Each route module should export `{ Component }` as a named export (not default export) for the lazy loading to work correctly. Alternatively, export a default `Component` function -- check React Router 7 docs for the exact convention. The safest pattern:
```typescript
// In each route file:
export function Component() { return <PageContent />; }
```
