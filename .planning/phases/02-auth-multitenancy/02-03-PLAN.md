Create 3 commands and 3 queries for tenant management following the CQRS pattern established in Phase 1. All handlers use `defineCommand`/`defineQuery` with TypeBox schemas and return `ok()`/`err()` results.

**Important:** These commands/queries wrap better-auth's organization plugin API. They do NOT use `ctx.db` (scopedDb) directly for org operations because auth/org tables are managed by better-auth, not by our scopedDb wrapper (per Pitfall 6). Instead, they call `auth.api.*` methods.

The `ctx` (HandlerContext) provides `tenantId` and `userId` for authorization checks. The actual org operations go through `auth.api`.

**1. packages/modules/auth/src/commands/create-tenant.ts:**
```typescript
import { Type } from "@sinclair/typebox";
import { defineCommand, ok, err } from "@baseworks/shared";
import { auth } from "../auth";

const CreateTenantInput = Type.Object({
  name: Type.String({ minLength: 1, maxLength: 100 }),
  slug: Type.Optional(Type.String({ minLength: 1, maxLength: 50, pattern: "^[a-z0-9-]+$" })),
});

export const createTenant = defineCommand(CreateTenantInput, async (input, ctx) => {
  try {
    const org = await auth.api.createOrganization({
      body: {
        name: input.name,
        slug: input.slug || input.name.toLowerCase().replace(/[^a-z0-9]+/g, "-").slice(0, 50),
      },
      headers: new Headers(),
    });
    ctx.emit("tenant.created", { tenantId: org.id, createdBy: ctx.userId });
    return ok(org);
  } catch (error: any) {
    return err(error.message || "Failed to create tenant");
  }
});
```

**1. Create packages/modules/auth/src/commands/update-profile.ts:**
Per D-17: User can update name, email, avatar URL, password.
```typescript
import { Type } from "@sinclair/typebox";
import { defineCommand, ok, err } from "@baseworks/shared";
import { auth } from "../auth";

const UpdateProfileInput = Type.Object({
  name: Type.Optional(Type.String({ minLength: 1, maxLength: 100 })),
  email: Type.Optional(Type.String({ format: "email" })),
  image: Type.Optional(Type.String()),
  currentPassword: Type.Optional(Type.String()),
  newPassword: Type.Optional(Type.String({ minLength: 8, maxLength: 128 })),
});

export const updateProfile = defineCommand(UpdateProfileInput, async (input, ctx) => {
  try {
    // Update basic profile fields via better-auth user update API
    const updateData: Record<string, any> = {};
    if (input.name !== undefined) updateData.name = input.name;
    if (input.email !== undefined) updateData.email = input.email;
    if (input.image !== undefined) updateData.image = input.image;

    if (Object.keys(updateData).length > 0) {
      await auth.api.updateUser({
        body: updateData,
        headers: new Headers(),
      });
    }

    // Password change requires separate API call
    if (input.newPassword && input.currentPassword) {
      await auth.api.changePassword({
        body: {
          currentPassword: input.currentPassword,
          newPassword: input.newPassword,
        },
        headers: new Headers(),
      });
    }

    return ok({ updated: true });
  } catch (error: any) {
    return err(error.message || "Failed to update profile");
  }
});
```

Note: The exact better-auth API for `updateUser` and `changePassword` may differ. Adapt method names to match actual API. The key contract: update user fields and optionally change password.

**2. Create packages/modules/auth/src/queries/get-profile.ts:**
```typescript
import { Type } from "@sinclair/typebox";
import { defineQuery, ok, err } from "@baseworks/shared";
import { auth } from "../auth";

const GetProfileInput = Type.Object({});

export const getProfile = defineQuery(GetProfileInput, async (_input, ctx) => {
  try {
    // Use the session to get current user data
    // The session already has user info from middleware,
    // but we query fresh data for accuracy
    const session = await auth.api.getSession({
      headers: new Headers(),
    });
    if (!session) return err("Not authenticated");
    return ok({
      id: session.user.id,
      name: session.user.name,
      email: session.user.email,
      image: session.user.image,
      emailVerified: session.user.emailVerified,
      createdAt: session.user.createdAt,
    });
  } catch (error: any) {
    return err(error.message || "Failed to get profile");
  }
});
```
